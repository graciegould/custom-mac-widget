# Custom Mac Widget

A simple macOS widget extension built with SwiftUI and WidgetKit. This widget displays the current time and is compatible with macOS Sonoma and later, using the required `.containerBackground(for: .widget)` modifier for desktop widgets.

## Features
- Displays the current time in a large font
- Supports small and medium widget sizes
- Uses the latest WidgetKit APIs for desktop compatibility

## Getting Started
1. Open the project in Xcode.
2. Build and run the main app at least once.
3. Add the widget from the system widget gallery.

## Requirements
- macOS Sonoma or later
- Xcode 15 or later

## Project Structure
- `MacWidget/` - Main app
- `MacWidgetExtension/` - Widget extension
